# Contributing to Jersey Votes

Contributions are welcome — bug reports, data corrections, and feature ideas all help improve this resource for Jersey residents.

## How to contribute

**Found a bug?** [Open an issue](https://github.com/gusfraser/jerseyvotes.org/issues/new?template=bug_report.md).

**Have a feature idea?** [Open an issue](https://github.com/gusfraser/jerseyvotes.org/issues/new?template=feature_request.md) to discuss it before building.

**Want to submit code?** Fork the repo, make your changes on a branch, and open a pull request. I review all PRs and have final say on what gets merged.

## Ground rules

- Keep PRs focused — one change per PR is easier to review
- Data accuracy matters: if you're changing how votes are calculated or displayed, explain your reasoning
- Be respectful in issues and reviews
